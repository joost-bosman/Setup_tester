# Contributing

Thank you for helping improve the Multilingual Developer Diagnostics Tool.

## Quick start
```bash
npm install
npm start
```

## Running checks
```bash
npm run smoke
npm run docs:pdf
```

## Build installers
```bash
npm run pack
npm run build
```

## Pull requests
- Keep PRs focused and small when possible.
- Include clear before/after notes and screenshots for UI changes.
- Update documentation if behavior or output changes.
- Follow existing code style and keep comments short and high-level.

## Issues
If you find a bug or have an idea, please use the GitHub issue templates.
